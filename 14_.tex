$\mathbf{Homework}$  $\mathbf{3.2.3.1}$
\\
Find the missing assignment to make the following programme segment correct.
\\
\\
$\left\{Q : 0 < i < n\right\}$
\\
$i, j := i + 1, ?$
\\
$\left\{R : j = n - 1 \land 0 \leq j < n \right\}$
\\
\\
Ans: $j := n - i - 1$
\\
\\
