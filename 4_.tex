\section{The Hoare Triple}
The Hoare triple
$\left\{Q\right\}$ $S$ $\left\{R\right\}$
holds if and only if $S$
, when executed
starting in a state that satisfies
predicate $Q$,
completes in a finite 
amount of time in a state
that satisfies predicate $R$.
\\
